# Smile Sanctuary

Providing exceptional dental care with a gentle touch since 2020. This project is a professional website for Smile Sanctuary dental clinics (D.Capital Dental Clinic & ZA Dental Clinic).

## Tech Stack
- **Framework**: React with Vite
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **UI Components**: shadcn/ui & Radix UI
- **Icons**: Lucide React

## Getting Started

1. **Clone the repository**:
   ```sh
   git clone <repository-url>
   ```

2. **Install dependencies**:
   ```sh
   npm install
   ```

3. **Run the development server**:
   ```sh
   npm run dev
   ```

## Key Features
- Modern, responsive design
- Gallery showcase (images only)
- Patient reviews and testimonials
- Location management for multiple clinics
- Appointment booking integration (WhatsApp, SMS, Email)
